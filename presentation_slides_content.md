## Speaker 2: System Architecture & Elite Design
*Focus: Tech stack, ERD, and the "SaaS-Ready" aesthetic.*

### Slide 6: System Scope
- **Concept:** End-to-End lifecycle management.
- **Modules:** Spot Discovery, Verification, Licensing, and Rent Compliance.

### Slide 7: Non-Functional Excellence
- **Security:** Supabase Row-Level Security (RLS) + Role-based access.
- **Performance:** React 19 SPA with high-concurrency map interactions.
- **Aesthetics:** **Premium "Vendor Hub" Design System.** (Clean, high-contrast, professional SaaS look).

### Slide 8: Database Design (ERD)
- **Hierarchy:** Zone -> Block -> Specific GPS Spot.
- **Identity:** Profiles (NID verified) linked to Permission Artifacts and Spot Assignments.
- **Impact:** Every merchant is a data point in a structured city registry.
